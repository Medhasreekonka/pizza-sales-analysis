# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
An interactive Pizza Sales Analysis project built using Python, SQL, Pandas, Plotly, Streamlit, and Power BI to analyze revenue trends, customer ordering behavior, and sales performance.

---

## 🚀 Technologies Used
- Python
- SQL
- Pandas
- Plotly
- Streamlit
- Power BI
- CSV Dataset

---

## 📊 Features
- Revenue and KPI analysis
- Daily and monthly sales trends
- Sales by pizza category and size
- Top & bottom-selling pizzas
- Revenue by hour analysis
- Interactive dashboard filters
- Download filtered dataset feature
- Business insights visualization

---

## 🛠 Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Development
- SQL Querying
- Business Intelligence
- Analytical Reporting

---

## 📈 Key Insights
- Classic pizza category generated maximum sales revenue
- Large-sized pizzas contributed highest sales
- Maximum orders occurred during weekends
- July and January recorded highest order volumes

---

## 📂 Project Files
- `pizza_sales.sql` → SQL queries and analysis
- `Pizza_sales_dashboard.py` → Streamlit dashboard application
- `pizza_sales_analysis.ipynb` → Jupyter Notebook analysis
- `pizza_sales.csv` → Dataset
- `pizza_sales_powerBi.pdf` → Power BI dashboard report

---

## ▶️ Run the Project

```bash
pip install streamlit pandas plotly
streamlit run Pizza_sales_dashboard.py
```
